# Resto App
## Описание проекта
Статичный веб-сайт ресторана проекта [Preax](https://preax.ru/) на классическом стеке без использования сборщиков, препроцессоров, библиотек и фреймворков.
## Стек технологий
HTML, CSS, JS
## Итог
**Получилось:** 
  
✅ Семантическая верстка  
✅ Применил теги time и address  
✅ SVG спрайт  
✅ Pixel Perfect  
✅ Декомпозиция CSS кода  
✅ БЭМ нейминг CSS классов  
✅ Изображения в формате webp, тег picture с jpg для поддержки  
✅ Доступность в виде aria атрибутов  

**Не получилось:**  
  
❌ Мало JS функционала, потому что базовый спринт, самый простой  
❌ Нет адаптива - не было в задачах спринта  
  
Проектом доволен. Закрепил навык семантической верстки. Поработал с svg (сделал спрайт, поправил fill на currentColor) и изображениями (сжал, переконвертировал в webp). Впервые верстал по Pixel Perfect. Добавил aria атрибуты для ссылок и кнопок без текста. Понял важную вещь, что работать с дедлайнами - вообще другая разработка. Очень понравилась платформа Preax тем, что дают макет и ТЗ. Не нужно выдумывать себе проект и функционал. В задачах самые популярные вещи, которые 100% встретятся на работе. Проекты не слишком большие, позволяют закрепить стек, не растягивая проекты на полгода. Ну и код ревью, впервые делал, сильно прокачивает, но все зависит от уровня ревьюера.
